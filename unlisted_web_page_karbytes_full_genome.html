karbytes_full_genome

Instructions on how to concatenate each of the "chunk" files in those "micro directories" into a single (Gzip-compressed) FASTA file are available in the plain-text journal entry files in the GitHub repository associated with the "micro directory" named raw_github_files_directory_karbytes_human_genome_fasta_part_0.
